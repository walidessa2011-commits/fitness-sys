# نظام إدارة النوادي والمراكز الرياضية (Fitness Club KSA)

نظام متكامل واحترافي لإدارة المراكز الرياضية والأندية، مبني باستخدام أحدث تقنيات الويب (Next.js & Supabase). يوفر النظام منصة شاملة لإدارة الاشتراكات، الأمور المالية، خدمة العملاء (CRM)، التذاكر، وتقارير تفصيلية لصناع القرار.

## 🌟 المميزات الأساسية (Features)

- 👥 **إدارة الأعضاء والاشتراكات:** إدارة مرنة للعملاء وسجلاتهم، تتبع فترات الاشتراكات، الإيقاف، التجديد، والتنبيهات التلقائية.
- 💰 **الإدارة المالية والخزينة:** تسجيل إيرادات الاشتراكات، إدارة المصروفات الداخلية، تقارير مالية، وتفصيل طرق الدفع (كاش، شبكات، تحويل).
- 🎫 **نظام التذاكر والدخول اليومي:** إصدار تذاكر يومية ومتابعتها.
- 📞 **خدمة العملاء (CRM):** متابعة طلبات العملاء، الرسائل، بالإضافة إلى التكامل مع الواتساب لإرسال الإشعارات والوسائط.
- 📊 **تحليل وتقارير البيانات:** لوحة تحكم إحصائية قوية مع القدرة على تصدير وطباعة التقارير بنسخ (PDF و Excel) بتصميم احترافي.
- 🔐 **صلاحيات وصول دقيقة:** أدوار مخصصة لكل مستخدم (مدير عام، مدير نادي، محاسب، موظف استقبال).
- 🌗 **واجهة مستخدم عصرية (UI/UX):** تصميم متجاوب يدعم الهواتف والأجهزة اللوحية (Responsive)، بالإضافة إلى الدعم الكامل للوضع الليلي والنهاري (Dark/Light mode) بلمسات عصرية و Micro-animations.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **إطار العمل (Framework):** [Next.js](https://nextjs.org/) (App Router)
- **لغة البرمجة:** TypeScript
- **تصميم الواجهات (Styling):** Tailwind CSS, Framer Motion
- **قواعد البيانات والاستضافة (Backend):** [Supabase](https://supabase.com/) (PostgreSQL & Auth)
- **أيقونات ومكونات:** Lucide React

## 🚀 كيفية تشغيل المشروع محلياً (Getting Started)

### 1- المتطلبات المسبقة:
- تثبيت [Node.js](https://nodejs.org/) (إصدار 18+).
- حساب من قاعدة بيانات Supabase.

### 2- خطوات التثبيت:

1. قم بإنشاء ملف `.env.local` في المجلد الرئيسي وضع فيه مفاتيح الربط الخاصة بـ Supabase:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

2. تثبيت الحزم (Dependencies):
   ```bash
   npm install
   ```

3. تشغيل خادم التطوير (Development Server):
   ```bash
   npm run dev
   ```

4. افتح المتصفح على [http://localhost:3000](http://localhost:3000) لاستعراض النظام الحى.

## 🔒 ملاحظات هامة للأمان
تأكد من عدم رفع ملف `.env.local` الخاص بك إلى مستودع GitHub لحماية مفاتيح API الخاصة بقواعد البيانات ومعلومات الخوادم. (مدعوم افتراضياً في `.gitignore`).
